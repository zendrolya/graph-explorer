# Graph Explorer

Учебный проект, находящийся в разработке и пока что имеющий только консольный интерфейс в браузере. Позволяет работать с графами:

1. Создавать графы с привязкой к названию.
2. Переключаться по графам, копировать и удалять их по названию, просматривать список доступных графов.
3. Сохранять граф в текстовый файл и загружать тестовые примеры из текстовых файлов.
4. Использовать методы графов, например, такие как: "вершины, не смежные с данной", "степень вершины", "сильно связные компоненты", "отображение списка смежности" и другие.

Реализован графический интерфейс для отображения графа на рабочем полотне.
