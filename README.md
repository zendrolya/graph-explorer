# Graph Explorer

Учебный проект, позволяющий работать с графами как в консольном, так и в графическом интерфейсе.

Ключевые возможности:

1. Создание графов с привязкой к названию.
2. Просматривание списка доступных графов и переключение по ним, копирование и удаление по названию.
3. Сохранение графа в текстовый файл и загрузка из текстового файла, загрузка готовых тестов из файлов-примеров.
4. Использование методов графов, например, таких как: "Вершины, не смежные с данной", "Степень вершины", "Сильно связные компоненты" и другие.
5. Пошаговая анимация метода "Максимальный поток".
